# 🎓 Course Registration System

A simple web-based Course Registration System developed using HTML, CSS, and JavaScript.

---

## 📌 Project Description

This system allows students to:
- View available courses
- Register for courses
- Drop registered courses
- Track total credits
- Enforce maximum credit limits

The project is designed as part of Software Engineering.

---

## 🚀 Features

- 🔐 Login System
- 📚 View Available Courses
- ✅ Register Courses
- ❌ Drop Courses
- 🎯 Credit Limit Validation
- 🔄 Real-time Credit Calculation
- 💾 LocalStorage for Data Persistence
- 🎨 Modern UI Design

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage (Browser Storage)
